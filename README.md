# r2dbc-demo

Project using Spring Boot with Webflux and Spring Data R2DBC and H2-R2DBC as a reactive database layer.

## Prerequisite
- Java 11
- Maven

## How to run
- Run `main` in `R2dbcDemoApplication`

## Endpoints
- **POST** `/customers`
```
{
  "firstname": "Bojan",
  "lastname": "Trajkovski"
}
```
